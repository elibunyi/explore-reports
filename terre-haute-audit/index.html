Priority 1 — Critical (Fix Immediately)

P1 Only 1 Ad Group — Entire Account Is a Single Bucket
The entire campaign uses a single ad group ("Ad group 1") with 21 keywords covering fiberglass pools, pool builders, pool contractors, pool installation, inground pools, and brand terms. This means every search — from "aviva pools" to "pool designs" to "swimming pool contractors" — triggers the same RSA ad with the same headlines and descriptions. Google cannot match ad copy to search intent because there's no segmentation. This is the #1 structural problem in this account and the root cause of the Average ad strength, inflated CPC, and low conversion rate.
P1 Ad Strength: Average (Should Be Good or Excellent)
The only RSA has "Average" ad strength. With only 1 ad group, Google cannot optimize headline/description combinations for different search intents. The headline category analysis below shows the distribution is credential-heavy and desire-light. This directly suppresses impression share and inflates CPC.
P1 5 of 9 Sitelinks Missing Description Lines
"Financing Options", "Pool Designs", "Range Of Pool Colors", "All Quality", and "Automatic Pool Covers" have zero description lines. Sitelinks without both Description Line 1 and Description Line 2 are ineligible for the expanded sitelink format (15–20% higher CTR). More than half the sitelinks are underperforming silently.
P1 Zero Negative Keywords — $236+ Wasted on Irrelevant Searches
The account has zero negative keywords. Search terms show paid clicks for "above ground pool" ($141+), "pool liner replacement" ($12.87), "concrete pool near me" ($9.32), "diy pool kits" ($5.69), "swim spa" terms, "pool service near me" ($6.89), "pool repair" terms, and competitor names like "sparkle pools" ($48.47 combined). Estimated waste: $236+ and climbing.
P1 Path 1 and Path 2 Both Blank on RSA
The display URL shows as avivapoolsterrehaute.com with no path context. Both Path 1 and Path 2 are empty. This wastes valuable ad real estate that signals relevance to searchers. Should be set to "Fiberglass-Pools" / context-specific path per ad group.

Priority 2 — High (Fix This Week)

P2 2 Duplicate Sitelink URL Destinations
"Discover Paradise at Home" and "Pool Designs" both go to /stylish-pool-designs/. "Range Of Pool Colors" and "Stunning Pool Colors" both go to /pool-colors/. Duplicate URLs reduce sitelink diversity and waste extension slots. Replace duplicates with unique landing pages (e.g., /about-us/, /gallery/, /contact/).
P2 Only 5 Unique Callouts (Recommend 8+)
10 callout rows but only 5 unique: "Passionate Pool Builders", "Aviva Fiberglass Pools", "Flexible Financing", "Lifetime Warranty", "Custom Pool Designs". Need at least 3 more unique callouts. Missing service-specific and trust-building callouts.
P2 Sitelink "All Quality" — Vague, Non-Descriptive Text
"All Quality" means nothing to a searcher scanning ads. Sitelink text should clearly communicate what the user will find. Rename to "Our Build Quality" or "Why Fiberglass?" and add description lines.
P2 Headline Pinning Reduces Google's Optimization Flexibility
3 headlines pinned to Position 1, 2 pinned to Position 2. Excessive pinning locks combinations and is a common cause of Average ad strength. With proper ad group segmentation, the Position 1 pins for geo-relevance become unnecessary because each ad group can have geo-specific keywords.
P2 $33/day Budget = $1,023/mo (Exceeds $1,000 Target)
Daily budget is set at $33.00, which works out to ~$1,023/month. At the current $8.11 avg CPC, this yields roughly 126 clicks/month. With the 1.88% conversion rate, that's ~2.4 conversions/month at $432 per conversion — very expensive for pool leads.
P2 Maximum CPC Bid Limit Set at $250
Campaign-level CPC cap is $250. While Maximize Clicks is the correct bid strategy for this account age (3 total conversions), the cap should be $15–$20 for pool dealer campaigns, not $250. This gives Google essentially no ceiling, which explains the $10+ and $20+ CPCs seen in the search terms report.
P2 No Structured Snippets
Zero structured snippets in the account. Should have at least "Models" (The Apex, The Triumph, The Dynasty, etc.) and "Amenities" (Tanning Ledge, Bench Seating, Deep End, etc.) snippets to increase ad real estate.

Priority 3 — Moderate (Optimize When Possible)

P3 21 Keywords in Single Ad Group (High Density)
With proper segmentation into 3–4 ad groups, keywords should be 8–15 per group. 21 in one bucket means intent dilution across fiberglass, contractor, installation, brand, and design terms.
P3 ZIP Code Targeting (47 ZIP Codes) — Consider Named Cities
Location targeting uses 47 individual ZIP codes across West Central Indiana. This works but named city/region targeting is easier to manage and audit. Verify all target ZIPs are within the dealer's actual service area — some show Reach: 0 (meaning no impressionable population in Google's estimate).
P3 "Financing Options" and "Flexible Pool Financing" Sitelinks — Redundant Concept
Two sitelinks about financing pointing to different URLs (/pool-financing/ vs /financing/). Consolidate to one strong financing sitelink and use the freed slot for a more diverse destination.
