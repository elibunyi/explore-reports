1. Systemic Issues (Affect Multiple Reports)

Issue A: Fabricated Pool Model Names in Structured Snippets

Severity: CRITICAL | Affects: 18 of 31 reports

The overnight audit sessions hallucinated pool model names when recommending structured snippets. The most common fabrication is "Monaco" -- it sounds like a plausible pool name but does NOT exist in any Explore brand. Other fabricated names follow a similar pattern: Italian/Mediterranean-sounding words that sound like they could be pool models but are not.

Root cause: The audit sessions did not load brand skill files before generating model name recommendations. They guessed model names based on the "luxury pool" association pattern.

Issue B: Sitelink URLs Pointing to Manufacturer Instead of Dealer

Severity: HIGH | Affects: ~12 reports

Multiple reports recommend sitelink URLs pointing to leisurepoolsusa.com paths (the manufacturer website) rather than the dealer's own website. These dealers have their own websites -- sitelinks should point there. Additionally, the URL paths used (e.g., /pools, /cost, /models) likely do not exist on leisurepoolsusa.com. The real site uses /fiberglass-swimming-pools/ for pool listings.

Issue C: Inflated Model Count Claims

Severity: MEDIUM | Affects: ~6 reports

Several reports include headlines like "Browse 50+ Pool Models" or "40+ Pool Models." Leisure Pools has exactly 31 pool models (plus 3 spas and 3 tanning ledges = 37 products total). "30+ Pool Models" is accurate; "50+" is not.

Issue D: "Palladium" and "Fiji" Used Without Full Name

Severity: LOW | Affects: ~5 reports

The correct model names are "Palladium Plunge" and "Fiji Plunge" -- not just "Palladium" or "Fiji." Using the truncated names in structured snippets would confuse customers and not match the brand's product pages.

Issue E: "Spirit" Is Not a Leisure Pools Model

Severity: HIGH | Affects: 4 reports (Houston, San Antonio, Sarasota, Pronto Pools)

"Spirit" appears in structured snippet recommendations for Leisure Pools dealers, but there is NO model called "Spirit" or "The Spirit" in the Leisure Pools lineup. This is a fabrication.

For Sitelink URLs

All sitelinks must point to the dealer's own website, not leisurepoolsusa.com. Pull the dealer's website from the Google Ads account's final_urls field or from the account's linked website.

For Model Count Claims

Replace "50+ Pool Models" or "40+ Pool Models" with "30+ Pool Models" for Leisure Pools dealers. If including spas and tanning ledges in the count, "35+ Products" is acceptable.

Prevention for Future Audits

  • Always load the brand skill file BEFORE generating model name recommendations
  • Never guess pool model names -- only use names from the verified brand registry
  • Sitelink URLs must come from the Google Ads account data (final_urls), not invented
  • Include a "verify models with dealer" disclaimer if the dealer's specific model lineup is unknown
  • Use generic snippet headers (Types, Amenities) when model names cannot be confirmed
