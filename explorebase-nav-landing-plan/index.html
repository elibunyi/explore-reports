Landing Page Template

Each ExploreBase page follows this layout. Section 1 uses generic copy that's the same across all pages.

ExploreBase — [City, State/Country]

[Hero image — use generic ExploreBase imagery initially, swap for location photos later]

Section 1: About ExploreBase

  • Generic copy about what ExploreBase facilities do — same across all pages
  • Role of ExploreBase in the Explore Industries network
  • How ExploreBase supports dealers in the region

Section 2: Service Area

  • States/provinces/regions this ExploreBase serves
  • Delivery radius or coverage map
  • Can potentially pull from territory maps for each ExploreBase

Section 3: Address & Contact

  • Full street address
  • Google Maps embed
  • Phone number
  • ExploreBase GM name + contact info

Section 4: Brands Available

  • Standard brand portfolio content block (same as used on other pages)

Section 5: CTA

  • "Become a Dealer" button
  • "Contact Us" form or link

Section 6: Image Gallery / Carousel

  • Generic ExploreBase imagery initially
  • Swap in personalized location photos as they become available

Part 3: Phased Rollout

Phase 1 — Build & Launch

  • Set up all 301 redirects from /locations/* to /explorebase/* (do this first)
  • Rebrand existing 4 pages (Knoxville, Marion, New Braunfels, Gold Coast)
  • Build new landing pages for all confirmed locations using template
  • Reorganize nav: rename "Locations" → "ExploreBase Locations" with single mega menu (all regions visible)
  • Move Global Development off dropdown → content block + link on /explorebase/ overview page
  • Consider moving Resources off main nav (content to footer or under Our Company)
  • Ensure all locations are reflected in nav and on overview page
  • Audit and update internal site links pointing to old /locations/ URLs
  • Submit updated sitemap to Google Search Console

Phase 2 — Personalized Copy

  • Write unique copy for each ExploreBase location
  • Define and publish service areas per location
  • Add GM names and direct contact info

Phase 3 — GM Photos & Maps

  • Get professional photos of each ExploreBase GM
  • Update service area maps with accurate coverage zones
  • Add Google Maps embeds where missing

Phase 4 — Location-Specific Imagery

  • Replace generic ExploreBase imagery with photos from each location
  • Hero images, facility photos, gallery/carousel content
  • This is the final polish step
